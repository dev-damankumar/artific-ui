# Buttons

Custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

export const defaultTemplate=(args) => <Button {...args}>Default</Button>;

<Canvas>
	<Story name="Default">{defaultTemplate.bind({})}</Story>
</Canvas>

## API

```jsx
import { Button } from 'artific-ui/Button';
```

## Themes

Use any of the available button themes types to quickly create a styled button. Just modify the `theme` prop.
