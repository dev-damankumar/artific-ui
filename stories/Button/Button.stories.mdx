# Buttons

Buttons allow users to take actions, and make choices, with a single tap.

## API

```jsx
import { Button } from 'artific-ui/Button';
```

## Button Themes

If you want to use a different theme for `Button` use the `theme` prop to set the theme
Note: All components have basic four themes ie. (`primary`|`secondary`|`dark`|`light`). you can use either of them.

<Canvas>
	<Story name="Button Themes">
		<Space>
			<Button theme="primary" type="button">Primary</Button>
			<Button theme="secondary" type="button">Secondary</Button>
			<Button theme="light" type="button">Light</Button>
			<Button theme="dark" type="button">Dark</Button>
		</Space>
	</Story>
</Canvas>

## Custom Theme Buttons

If you want to use a different color scheme for `Button` instead of the provided themes then you can use
`colorScheme` props to define your color scheme.

colorScheme is an object containing three properties: `background`, `backgroundColor`, `color`

<Canvas>
	<Story name="Custom theme buttons">
		<Space>
			<Button colorScheme={{
				background: 'linear-gradient(45deg, #e736f4, #673ab7)',
				backgroundColor: '#e736f4',
				color: 'white',
			}} type="button">Primary</Button>
			<Button colorScheme={{
				background: 'red',
				color: 'white',
			}} theme="light" type="button">Light</Button>
			<Button colorScheme={{
				background: 'linear-gradient(45deg, #e736f4, #673ab7)',
				backgroundColor: '#e736f4',
				color: 'white',
			}} variant="outline" type="button">Custom Outline</Button>
			<Button colorScheme={{
				background: 'red',
				color: 'white',
			}} variant="outline" theme="light" type="button">Custom Outline</Button>
		</Space>
	</Story>
</Canvas>

## Variants

Artific UI provides a `variant` prop to make you choose how your component ui should look like

Set `variant` prop with the value of (`default`|`text`|`outline`). you can mix them with different supported `theme` or `layout`

Note: Different components might have addition or different variant based on there configuration and flexibility.
you can check out the props table to see which variant is supported by the component

<Canvas>
	<Story name="Variants">
		<Space>
			<Button variant="default" type="button">Default</Button>
			<Button variant="text" type="button">Text</Button>
			<Button variant="outline" type="button">Outline</Button>
		</Space>
	</Story>
</Canvas>

## Icons In Buttons

You can append or prepend an icon to the button with prop called `prefix` and `suffix`

any jsx is allowed in these props.

<Canvas>
	<Story name="Icons In Buttons">
		<Space>
			<Button
				prefix={<i className='bx bxs-user-circle'></i>}
				type="button"
			>User
			</Button>
			<Button theme="light" type="button" suffix={<i className='bx bx-cog'></i>}>Settings</Button>
			<Button theme="dark"
					prefix={

			<Button theme="light" type="button"
					suffix={<img

		</Space>
	</Story>

Set `layout` prop with the value of (`default`|`rounded`|`no-radius`|`pill`)

`Note:` Different components might have addition layouts based on there configuration and flexibility. you can check out the props table to see which layout is supported by the component

## Sizes

you can use a `size` prop to define the size of your components

Set `size` prop with the value of (`xs`|`sm`|`md`|`lg`,`xl`). you can mix them with different supported `theme` or `layout`

Note: Mostly all the components have the same size values ie.(`xs`|`sm`|`md`|`lg`,`xl`). but some of the components might have additional sizes.

## Loading Buttons

When you click the button you can set it to loading state with the prop called `loading`

also you can choose to hide the text of button when `loading` is true, by setting with the `hideTextOnLoading` props

also you can use `loadingStyle:` (`spin`|`grow`) to change the loading style.

use `loadingDirection` to set on with direction loader should appare.

<Canvas>
	<Story name="Loading Buttons">
		<Space>
			<Button loading hideTextOnLoading loadingStyle="spin" type="button">Loading Grow</Button>
			<Button loading hideTextOnLoading loadingStyle="grow" type="button">Loading Grow</Button>
			<Button loading loadingStyle="grow" type="button">Loading Grow</Button>
			<Button loading loadingDirection="after" type="button">Loading Spin</Button>
			<Button loading loadingDirection="after" loadingStyle="grow" type="button">Loading Grow</Button>
		</Space>
	</Story>
</Canvas>
