# IconButtons

Custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

export const defaultTemplate=(args) => <IconButton {...args}>Default</IconButton>;

## API

```jsx
import { IconButton } from 'artific-ui/IconButton';
```
