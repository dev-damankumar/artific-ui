# Loaders

useLoader is used to show that a process is happening on the background and tells user to wait.

<div>
	<div name="Loaders">{Template.bind({})}</div>
</div>

## API

```jsx
import { useLoader } from 'artific-ui';
```

<b>useLoader()</b> instance has following methods to use:

`config()` to set the config and supported props to the loader.

`show()` method is used to show the loader.

`hide()`  method is used to hide the loader.

`destroy()`  method is used to remove the loader from the DOM.


export const DissmissableTemplate=(args) => {
	const message=useLoader();
	message.config({dismissible: true})
	return <div>
		<Button onClick={() => {
			message.show('Hi ! This is a message')
		}}>
			Loader
		</Button>
		<br/>
	</div>
};

# Dismissible

Dismissible Loader can be used to close the loader when user clicks on it.

provide `{dismissible: true}` to the `config` method when instantiating loader.

<div>

</div>

# Loader Themes

If you want to use a different theme for `Loader` use the `theme` prop in `config` method to set the theme

Note: Loader only have basic twi themes ie. (`dark`|`light`). you can use either of them.

<div>
	<div name="Loader Themes">{ThemeTemplate.bind({})}</div>
</div>

# Variants

Artific UI provides a `variant` prop to make you choose how your component ui should look like

Set `variant` prop with the value of (`default`|`blur`|`solid`). you can mix them with different supported `theme`

Note: Different components might have addition or different variant based on there configuration and flexibility.
you can check out the props table to see which variant is supported by the component

<div>
	<div name="Variants">{VariantsTemplate.bind({})}</div>
</div>

# Sizes

you can use a `size` prop to define the size of your components

Set `size` prop with the value of (`xs`|`sm`|`md`|`lg`,`xl`). you can mix them with different supported `theme` or `layout`

Note: Mostly all the components have the same size values ie.(`xs`|`sm`|`md`|`lg`,`xl`). but some of the components might have additional sizes.
