# Card

Cards can be used for headers and footers, a wide variety of content, contain contextual background colors and images

## Base Card

`Card` component consist of sub-components : `Card.Img`, `Card.Body`, `Card.Link`, `Card.Title`, `Card.Subtitle`, `Card.Text`, `Card.Header`, `Card.Footer`, `Card.ImgOverlay`.

<Canvas>
	<Story name="Base Card">{Template.bind({})}</Story>
</Canvas>

## API

```jsx
import { Card } from 'artific-ui/Card';
```

<ArgsTable story="Base Card"/>

SGDS `Card` component is build on top of React-bootstrap.
Please refer to https://react-bootstrap.github.io/components/cards/#api for more detailed API documentation

## Basic Cards Images Layout

Use `Card.StretchedLink` instead of `Card.Link` to "stretched" the nested link and have the entire `Card` component clickable.
Multiple links and tap targets are not recommended with stretched links.
