# ButtonGroup

You can group together multiple button with `ButtonGroup`

export const defaultTemplate=(args) => <ButtonGroup {...args}>
	<Button type="button">Primary</Button>
	<Button type="button">Light</Button>
	<Button type="button">Dark</Button>
</ButtonGroup>;

<div>
	<div name="Default">{defaultTemplate.bind({})}</div>
</div>

## API

```jsx
import { Button } from 'artific-ui/Button';
```



## Button Horizontal Group

Use `direction` props on `ButtonGroup` to define the direction of the ButtonGroup

default is `horizontal`

<div>
	<div name="Button Horizontal Group">
		<ButtonGroup>
			<Button type="button">Primary</Button>
			<Button theme="secondary" type="button">Secondary</Button>
			<Button theme="light" type="button">Light</Button>
			<Button theme="dark" type="button">Dark</Button>
		</ButtonGroup>
	</div>
</div>


## Button Vertical Group

Use `direction` props on `ButtonGroup` to define the direction of the ButtonGroup

default is `horizontal`

<div>
	<div name="Button Vertical Group">
		<ButtonGroup direction="vertical">
			<Button type="button">Primary</Button>
			<Button theme="light" type="button">Light</Button>
			<Button theme="dark" type="button">Dark</Button>
		</ButtonGroup>
	</div>
</div>

## Button Group Sizes

you can use a `size` prop to define the size of your components

Set `size` prop with the value of (`xs`|`sm`|`md`|`lg`,`xl`). you can mix them with different supported `theme` or `layout`

`Note:` when using `ButtonGroup` you don't have to provide the `size` prop on `Button` itself just use the props on the main component.
