# ButtonGroup

Custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

export const defaultTemplate=(args) => <ButtonGroup {...args}>
	<Button type="button">Primary</Button>
	<Button type="button">Light</Button>
	<Button type="button">Dark</Button>
</ButtonGroup>;

<Canvas>
	<Story name="Default">{defaultTemplate.bind({})}</Story>
</Canvas>

## API

```jsx
import { Button } from 'artific-ui/Button';
```

<ArgsTable story="Default"/>

## Button Horizontal Group

Use any of the available button themes types to quickly create a styled button. Just modify the `theme` prop.

<Canvas>
	<Story name="Button Horizontal Group">
		<ButtonGroup>
			<Button type="button">Primary</Button>
			<Button theme="light" type="button">Light</Button>
			<Button theme="dark" type="button">Dark</Button>
		</ButtonGroup>
	</Story>
</Canvas>
