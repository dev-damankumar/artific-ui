# Typography

Use typography to present your design and content as clearly and efficiently as possible.

export const defaultTemplate=(args) => <Typography {...args}>Lorem ipsum dolor sit amet.</Typography>;

<div>
	<div name="Default">{defaultTemplate.bind({})}</div>
</div>

## API

```jsx
import { Typography } from 'artific-ui/Typography';
```



## Default Typography

Use typography to present your design and content as clearly and efficiently as possible.

you can use `as` props to define which element type your typography should have.

<div>
	<div name="Default Typography">
		<Typography>Default Paragraph</Typography>
		<Typography as="span">Paragraph as span</Typography>
		<Typography as="div">Paragraph as div</Typography>
	</div>
</div>

## Typography Weights

Use `weight` prop with value (`light`|`medium`|`bold`) to set the font weight.

<div>
	<div name="Typography Weights">
		<Typography weight="light">Light Weight Paragraph</Typography>
		<Typography weight="medium">Medium Weight Paragraph</Typography>
		<Typography weight="bold">Bold Weight Paragraph</Typography>
	</div>
</div>

## Typography As HTML Element

Use `as` prop with value (`p`|`b`|`span`|`i`|`div`|`h1-h6`) to set the element type.

<div>
	<div name="Typography As HTML Element">
		<Typography as="p">Typography as `p`</Typography>
		<Typography as="b">Typography as `b`</Typography>
		<Typography as="span">Typography as `span`</Typography>
		<Typography as="i">Typography as `i`</Typography>
		<Typography as="div">Typography as `div`</Typography>
		<Typography as="h1">Typography as `h1`</Typography>
		<Typography as="h2">Typography as `h2`</Typography>
		<Typography as="h3">Typography as `h3`</Typography>
		<Typography as="h4">Typography as `h4`</Typography>
		<Typography as="h5">Typography as `h5`</Typography>
		<Typography as="h6">Typography as `h6`</Typography>
	</div>
</div>


## Variants

Artific UI provides a `variant` prop to make you choose how your component ui should look like

Set `variant` prop with the value of (`subtitle`|`heading`|`caption`|`description`|`highlight`|`abbr`|`kbd`|`pre`|`code`). you can mix them with different supported `theme` or `layout`

Note: Different components might have addition or different variant based on there configuration and flexibility.
you can check out the props table to see which variant is supported by the component

## Typography Themes


If you want to use a different theme for `Alert` use the `theme` prop to set the theme
Note: All components have basic four themes ie. (`primary`|`secondary`|`dark`|`light`). you can use either of them.

You can also use `colorScheme` prop to define a custom theme.

<div>
	<div name="Typography Themes">
		<Typography theme="primary">Default Paragraph</Typography>
		<Typography theme="secondary">Default Paragraph</Typography>
		<Typography style={{backgroundColor: 'black'}} theme="light">Default Paragraph</Typography>
		<Typography theme="dark">Default Paragraph</Typography>
		<Typography colorScheme={{
			background: 'linear-gradient(45deg, #e736f4, #673ab7)',
			backgroundColor: '#e736f4',
			color: "red",
		}}>Default Paragraph</Typography>
	</div>
</div>
