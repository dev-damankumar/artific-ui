## API

```jsx
import { Avatar } from 'artific-ui/Avatar';
```

## Avatar Themes

If you want to use a different theme for `Avatar` use the `theme` prop to set the theme
Note: All components have basic four themes ie. (`primary`|`secondary`|`dark`|`light`). you can use either of them.

You can also use `colorScheme` prop to define a custom theme.

<Canvas>
	<Story name="Avatar Themes">
		<Avatar theme="primary" title="DK"/>
		<Avatar theme="secondary" title="DK"/>
		<Avatar theme="dark" title="DK"/>
		<Avatar theme="light" title="DK"/>
		<Avatar colorScheme={{
			background: 'linear-gradient(45deg, #e736f4, #673ab7)',
			backgroundColor: '#e736f4',
			color: 'white',
		}} title="DK"/>
	</Story>
</Canvas>

## Avatar Size

you can use a `size` prop to define the size of your components

Set `size` prop with the value of (`xs`|`sm`|`md`|`lg`,`xl`). you can mix them with different supported `theme` or `layout`

Note: Mostly all the components have the same size values ie.(`xs`|`sm`|`md`|`lg`,`xl`). but some of the components might have additional sizes.

<Canvas>
	<Story name="Avatar Size">
		<Avatar size="xl" title="XL"/>
		<Avatar size="lg" title="LG"/>
		<Avatar size="md" title="MD"/>
		<Avatar size="sm" title="SM"/>
		<Avatar size="xs" title="XS"/>
	</Story>
</Canvas>

## Avatar Layouts

Avatar Has basic four layouts to choose from to define how you component's structure should look like.

Set `layout` prop with the value of (`default`|`rounded`|`no-radius`|`pill`)

`Note:` Different components might have addition layouts based on there configuration and flexibility. you can check out the props table to see which layout is supported by the component

<Canvas>
	<Story name="Avatar Layouts">
		<Avatar src={a1} layout="default"/>
		<Avatar src={a2} layout="rounded"/>
		<Avatar src={a3} layout="no-radius"/>
		<Avatar src={a4} layout="pill"/>
	</Story>
</Canvas>


## Avatar Variants

Artific UI provides a `variant` prop to make you choose how your component ui should look like

Set `variant` prop with the value of (`default`|`fluid`|`outline`). you can mix them with different supported `theme` or `layout`

Note: Different components might have addition or different variant based on there configuration and flexibility.
you can check out the props table to see which variant is supported by the component

<Canvas>
	<Story name="Avatar Variants">
		<Avatar src={a1} variant="default"/>
		<Avatar src={a2} variant="outline"/>
		<Avatar src={a3} variant="fluid"/>
	</Story>
</Canvas>

## Avatar Group

Avatar Group is used to group together a multiple avatars

Use `AvatarGroup` with `Avatar` as children to group them together
