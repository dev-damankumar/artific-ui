# Spaces

A Space Helps you to set the spacing of its children in a flexible way. It Avoids components clinging together and set a unified space.

By Default all the items will wrap if content overflows to make it not wrap the items set `wrap` prop to `false`
<Canvas>
	<Story name="Default">{Template.bind({})}</Story>
</Canvas>

## API

```jsx
import { Space } from 'artific-ui/Space';
```

<ArgsTable story="Default"/>

## Space Gap

A `gap` prop is used to define how much spacing the children should have.

`gap` prop value can be any number donating the px unit to define gap or it can be a tuple [number,number]

donation `horizontal` spacing and `verticle` spacing respectively

## Space Direction

A `direction` prop is used to define on which direction the spacing should be applied. by default it applies on `both`

`direction` prop values can be : (`horizontal`,`verticle`,`both`)

## Space Align

A `align` prop is used to align items on y-axis. its is same as the `align-item` css property

`align` prop value can be: (`start`|`end`|`baseline`|`center`)
