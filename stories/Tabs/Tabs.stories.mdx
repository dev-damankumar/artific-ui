# Tabs

Tabs make it easy to explore and switch between different views.

export const defaultTemplate=(args) => {
	return <Tabs {...args}>
		<TabItem active target="home">Home</TabItem>
		<TabItem target="about">About</TabItem>
		<TabContent id="home">
			<h3>HOME</h3>
			<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut
				labore
				et dolore magna aliqua.</p>
		</TabContent>
		<TabContent id="about">
			<h3>About</h3>
			<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut
				labore
				et dolore magna aliqua.</p>
		</TabContent>
	</Tabs>
};

## API

```jsx
import { Tabs } from 'artific-ui/Tabs';
```

## Tab Themes

If you want to use a different theme for `Tabs` use the `theme` prop to set the theme
Note: All components have basic four themes ie. (`primary`|`secondary`|`dark`|`light`). you can use either of them.

You can also use `colorScheme` prop to define a custom theme.

Artific UI provides a `variant` prop to make you choose how your component ui should look like

Set `variant` prop with the value of (`default`|`separate`|`line`|`floating-line`). you can mix them with different supported `theme` or `layout`

Note: Different components might have addition or different variant based on there configuration and flexibility.
you can check out the props table to see which variant is supported by the component
