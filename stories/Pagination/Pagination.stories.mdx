# Pagination

The Pagination component enables the user to select a specific page from a range of pages.

export const defaultTemplate=(args) => <Pagination {...args}>
	<PaginationItem>Prev</PaginationItem>
	<PaginationItem active>1</PaginationItem>
	<PaginationItem>2</PaginationItem>
	<PaginationItem>3</PaginationItem>
	<PaginationItem>Next</PaginationItem>
</Pagination>;

<div>
	<div name="Default">{defaultTemplate.bind({})}</div>
</div>

## API

```jsx
import { Pagination } from 'artific-ui/Pagination';
```



## Pagination Theme

If you want to use a different theme for `Pagination` use the `theme` prop to set the theme
Note: All components have basic four themes ie. (`primary`|`secondary`|`dark`|`light`). you can use either of them.

You can also use `colorScheme` prop to define a custom theme.
