# Modals

The Modal component is a dialog box/popup window that is displayed on top of the current page:


export const defaultTemplate=(args) => {

		<Button size="sm" type="button" onClick={() => {

			setOpen(false)
		}}>

## API

```jsx
import {Modal} from 'artific-ui/Modal';
```



## Dismissible

Dismissible Modal will close itself when user clicks outside of it.

`onClose` prop expect a callback which it will use to call when user click on the close button on header.

use `dismiss={false}` when you don't want user to close modal. it will remove the close button from header.

if you want user to close button when he clicks outside use `onBackdropClick` and provide it with callback to close the modal.

export const dismissTypesTemplate=(args) => {
	const [open, setOpen]=useState(false);
	const [open1, setOpen1]=useState(false);
	const [open2, setOpen2]=useState(false);

				setOpen2(true)

				setOpen1(true)

		<Modal
			open={open2}

Set `variant` prop with the value of (`default`|`striped`|`striped-body`|`striped-header`|`striped-footer`). you can mix them with different supported `theme` or `layout`

Note: Different components might have addition or different variant based on there configuration and flexibility.
you can check out the props table to see which variant is supported by the component
