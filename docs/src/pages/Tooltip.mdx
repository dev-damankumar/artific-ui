import { Button, Tooltip } from 'artific-ui'

# Tooltip

Tooltips display informative text when users hover over, focus on, or tap an element.

export const defaultTemplate = args => (
    <Tooltip title="This is tooltip">
        <Button>Default Tooltip</Button>
    </Tooltip>
)

<div>
    <div name="Default">{defaultTemplate.bind({})}</div>
</div>

## API

```jsx
import { Tooltip } from 'artific-ui/Tooltip'
```

## Tooltip Theme

If you want to use a different theme for `Tooltip` use the `theme` prop to set the theme
Note: All components have basic four themes ie. (`primary`|`secondary`|`dark`|`light`). you can use either of them.

You can also use `colorScheme` prop to define a custom theme.

<div>
    <div name="Tooltip Theme">
        <Tooltip title="This is tooltip">
            <Button>Default Tooltip</Button>
        </Tooltip>
        <br />
        <br />
        <Tooltip theme="secondary" title="This is tooltip">
            <Button>Secondary Tooltip</Button>
        </Tooltip>
        <br />
        <br />
        <Tooltip theme="dark" title="This is tooltip">
            <Button>Dark Tooltip</Button>
        </Tooltip>
        <br />
        <br />
        <Tooltip theme="light" title="This is tooltip">
            <Button>Light Tooltip</Button>
        </Tooltip>
        <br />
        <br />
        <Tooltip
            colorScheme={{
                background: 'linear-gradient(45deg, #e736f4, #673ab7)',
                backgroundColor: '#e736f4',
                color: 'white',
            }}
            title="This is tooltip"
        >
            <Button>Light Tooltip</Button>
        </Tooltip>
        <br />
        <br />
    </div>
</div>

## Tooltip Layouts

Tooltip Has basic four layouts to choose from to define how you component's structure should look like.

Set `layout` prop with the value of (`default`|`rounded`|`no-radius`|`pill`)

`Note:` Different components might have addition layouts based on there configuration and flexibility. you can check out the props table to see which layout is supported by the component

<div>
    <div name="Tooltip Layouts">

            <Button>Rounded Tooltip</Button>

## Tooltip Variants

Artific UI provides a `variant` prop to make you choose how your component ui should look like

Set `variant` prop with the value of (`default`|`arrow`). you can mix them with different supported `theme` or `layout`

Note: Different components might have addition or different variant based on there configuration and flexibility.
you can check out the props table to see which variant is supported by the component

<div>
    <div name="Tooltip Variants">
        <Tooltip variant="default" title="This is tooltip">
            <Button>Default Tooltip</Button>
        </Tooltip>
        <br />
        <br />
        <Tooltip variant="arrow" title="This is tooltip">
            <Button>Arrow Tooltip</Button>
        </Tooltip>
    </div>
</div>

## Tooltip Sizes

you can use a `size` prop to define the size of your components

Set `size` prop with the value of (`xs`|`sm`|`md`|`lg`,`xl`). you can mix them with different supported `theme` or `layout`

Note: Mostly all the components have the same size values ie.(`xs`|`sm`|`md`|`lg`,`xl`). but some of the components might have additional sizes.
