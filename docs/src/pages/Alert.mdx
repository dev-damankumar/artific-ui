import {useState} from 'react';
import {Alert, Button} from '../../../src/index';

export const Template=(args) => {
	const [show, setShow]=useState(true);
	if (show) {
		return (
			<Alert {...args} onClose={() => setShow(false)}>
				<i className="bi bi-info-circle flex-shrink-0 me-4"></i>
				This is an Alert
			</Alert>
		);
	}
	return <Button onClick={() => setShow(true)}>Show Alert</Button>;
};

# Alerts

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.

<div>
	<div name="Default">{Template.bind({})}</div>
</div>

## API

```jsx
import { Alert } from 'artific-ui/Alert';
```


## Dismissible Alert

A dismissible alert is a alert which can be removed from the user's screen by clicking on the close icon button

<div>
	<div name="Dismissible Alert">
		<Alert dismiss="true"><strong>Primary!</strong> You should <a href="data/Alert.stories#">read this
			message</a>.</Alert>
	</div>
</div>

## Alert Themes

If you want to use a different theme for `Alert` use the `theme` prop to set the theme
Note: All components have basic four themes ie. (`primary`|`secondary`|`dark`|`light`). you can use either of them.

<div>
	<div name="Alert Themes">
		<h5>Basic Alerts</h5>
		<br/>
		<Alert dismiss={true}><strong>Primary!</strong> You should <a href="data/Alert.stories#">read this
			message</a>.</Alert>
		<Alert dismiss={true} theme="secondary"><strong>Secondary!</strong> You should <a href="data/Alert.stories#">read
			this message</a>.</Alert>
		<Alert dismiss={true} theme="light"><strong>Light!</strong> You should <a href="data/Alert.stories#">read
			this
			message</a>.</Alert>
		<Alert dismiss={true} theme="dark"><strong>Dark!</strong> You should <a href="data/Alert.stories#">read this
			message</a>.</Alert>
		<br/>
	</div>
</div>

## 2. Custom Alert

If you want to use a different color scheme for `Alert` instead of the provided themes then you can use
`colorScheme` props to define your color scheme.

colorScheme is an object containing three properties: `background`, `backgroundColor`, `color`

<div>
	<div name="Custom Alert">
		<h5>CustomScheme Alerts</h5>
		<br/>
		<Alert dismiss="true"
			   colorScheme={{
				   background: 'linear-gradient(45deg, #e736f4, #673ab7)',
				   backgroundColor: '#e736f4',
				   color: 'white',
			   }}>
			<strong>Primary!</strong> You should <a href="data/Alert.stories#">read this message</a>.
		</Alert>
	</div>
</div>

## 3. Alert Sizes

you can use a `size` prop to define the size of your components

Set `size` prop with the value of (`xs`|`sm`|`md`|`lg`,`xl`). you can mix them with different supported `theme` or `layout`

Note: Mostly all the components have the same size values ie.(`xs`|`sm`|`md`|`lg`,`xl`). but some of the components might have additional sizes.

<div>
	<div name="Alert Sizes">
		<Alert size="xs">You should <a href="data/Alert.stories#">read this message</a>.</Alert>
		<Alert size="sm">You should <a href="data/Alert.stories#">read this message</a>.</Alert>
		<Alert>You should <a href="data/Alert.stories#">read this message</a>.</Alert>
		<Alert size="lg">You should <a href="data/Alert.stories#">read this message</a>.</Alert>
		<Alert size="xl">You should <a href="data/Alert.stories#">read this message</a>.</Alert>
	</div>
</div>

## 3. Alert Layouts

Alert Has basic four layouts to choose from to define how you component's structure should look like.

Set `layout` prop with the value of (`default`|`rounded`|`no-radius`|`pill`)

`Note:` Different components might have addition layouts based on there configuration and flexibility. you can check out the props table to see which layout is supported by the component

<div>

			message</a>.</Alert>

## 3. Alert Variants

Artific UI provides a `variant` prop to make you choose how your component ui should look like

Set `variant` prop with the value of (`default`|`text`|`note`|`outline`). you can mix them with different supported `theme` or `layout`

Note: Different components might have addition or different variant based on there configuration and flexibility.
you can check out the props table to see which variant is supported by the component

<div>
	<div name="Alert Variants">
		<Alert variant="outline"><strong>Outline!</strong> You should <a href="data/Alert.stories#">read this
			message</a>.</Alert>
		<Alert variant="default"><strong>Default!</strong> You should <a href="data/Alert.stories#">read
			thismessage</a>.</Alert>
		<Alert variant="note"><strong>Note!</strong> You should <a href="data/Alert.stories#">read this
			message</a>.</Alert>
	</div>
</div>

## 3. Note Variant Directions

Alert Note Variant has `direction` prop to define which direction the alert should follow.

Set `direction` prop with the value of (`top`|`bottom`|`left`|`right`). you can mix them with different supported `theme` or `layout`

<div>
	<div name="Note Variant Directions">
		<Alert variant="note">You should <a href="data/Alert.stories#">read this message</a>.</Alert>
		<Alert direction="right" variant="note">You should <a href="data/Alert.stories#">read this
			message</a>.</Alert>
		<Alert direction="top" variant="note">You should <a href="data/Alert.stories#">read this message</a>.</Alert>
		<Alert direction="bottom" variant="note">You should <a href="data/Alert.stories#">read this
			message</a>.</Alert>

</div>
