# Divider

The divider is used to divide section by putting line between them.

<div>
    <div name="Default">{Template.bind({})}</div>
</div>

## API

```jsx
import { Divider } from 'artific-ui'
```

## Divider Direction

The Divider has main two directions ie. (`horizontal`|`vertical`)

<div>
    <div name="Containers">
        <div style={{ display: 'flex', height: 200 }}>
            <div style={{ width: 200, background: 'gainsboro' }}>
                Basic Divider
            </div>
            <Divider direction="vertical" />
            <div style={{ width: 200, background: 'gainsboro' }}>
                Basic Divider
            </div>
        </div>
    </div>
</div>
