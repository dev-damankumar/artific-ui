## Accordion

The accordion component allows the user to show and hide sections of related content on a page.

An accordion is a lightweight container that may either be used standalone, or be connected to a larger surface, such as a card.

## Basic Accordion

<Accordion>
    <AccordionPanel>
        <AccordionHeader>Accordion Header</AccordionHeader>
        <AccordionBody>
            Lorem ipsum dolor sit amet, consectetur adipisicing elit. Aliquid
            debitis dolorem
        </AccordionBody>
    </AccordionPanel>
    <AccordionPanel>
        <AccordionHeader>Accordion Header 2</AccordionHeader>
        <AccordionBody>
            Lorem ipsum dolor sit amet, consectetur adipisicing elit. Aliquid
            debitis dolorem
        </AccordionBody>
    </AccordionPanel>
    <AccordionPanel>
        <AccordionHeader>Accordion Header 3</AccordionHeader>
        <AccordionBody>
            Lorem ipsum dolor sit amet, consectetur adipisicing elit. Aliquid
            debitis dolorem
        </AccordionBody>
    </AccordionPanel>
</Accordion>

## API

```jsx
import { Accordion } from 'artific-ui/Accordion'
```

## Accordion Custom Themes

If you want to use a different color scheme for `accordion` instead of the provided themes then you can use
`colorScheme` props to define your color scheme.

colorScheme is an object containing three properties: `background`, `backgroundColor`, `color`

<div name="Accordion Custom Themes">
    <Accordion
        colorScheme={{
            background: 'linear-gradient(45deg, #e736f4, #673ab7)',
            backgroundColor: '#e736f4',
            color: 'white',
        }}
        theme="primary"
    >
        <AccordionPanel>
            <AccordionHeader>Accordion Header</AccordionHeader>
            <AccordionBody>
                Lorem ipsum dolor sit amet, consectetur adipisicing elit.
                Aliquid debitis dolorem
            </AccordionBody>
        </AccordionPanel>
    </Accordion>
</div>

## Accordion Disabled Panel

If you want to disable a specific panel of the Accordion use `disabled` prop on `AccordionPanel` to disable it.

<div name="Accordion Disabled Panel">
    <Accordion theme="primary">
        <AccordionPanel disabled={true}>
            <AccordionHeader>Secret Header</AccordionHeader>
            <AccordionBody>This is a secret content</AccordionBody>
        </AccordionPanel>
    </Accordion>
    <Accordion theme="secondary">
        <AccordionPanel disabled={true}>
            <AccordionHeader>Secret Header</AccordionHeader>
            <AccordionBody>This is a secret content</AccordionBody>
        </AccordionPanel>
    </Accordion>
    <Accordion theme="light">
        <AccordionPanel disabled={true}>
            <AccordionHeader>Secret Header</AccordionHeader>
            <AccordionBody>This is a secret content</AccordionBody>
        </AccordionPanel>
    </Accordion>
    <Accordion theme="dark">

            <AccordionHeader>Secret Header</AccordionHeader>
            <AccordionBody>This is a secret content</AccordionBody>
        </AccordionPanel>
    </Accordion>
    <Accordion
        colorScheme={{
            background: 'linear-gradient(45deg, #e736f4, #673ab7)',

</div>

Note: Mostly all the components have the same size values ie.(`xs`|`sm`|`md`|`lg`,`xl`). but some of the components might have additional sizes.
