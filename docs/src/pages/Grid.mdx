export const colStyle={
	border: "1px dashed #089b8d",
	background: '#afe0db'
}



# Grids

The grid creates visual consistency between layouts while allowing flexibility across a wide variety of designs. Material Design's responsive UI is based on a 12-column grid layout.

<div>
	<div>
		<h5>Basic Layout</h5>
		<Container style={{background: '#fff0eb', padding: '20px 35px'}}>
			<Row style={{background: '#c2c3f5', padding: '20px'}}>
				<Col style={{...colStyle}}>Col 1</Col>
				<Col offset={3} style={{...colStyle}}>Col 2</Col>
				<Col style={{...colStyle}}>Col 3</Col>
			</Row>
		</Container>
	</div>
</div>

## API

```jsx
import { Container,Row,Col } from 'artific-ui';
```

## Containers

The container centers your content horizontally. It's the most basic layout element.

<div>
	<div name="Containers">
		<Container style={{background: '#f1f1f1', padding: '20px', height: 100}}>
			<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Asperiores consectetur et facere maxime
				molestiae omnis porro praesentium quisquam recusandae reprehenderit.</p>
		</Container>
	</div>
</div>


## Container Fluid

A fluid container width is bounded by the 100% screen width

<div>
	<div name="Container Fluid">
		<Container fluid={true} style={{background: '#f1f1f1', padding: '20px', height: 100}}>
			<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Asperiores consectetur et facere maxime
				molestiae omnis porro praesentium quisquam recusandae reprehenderit.</p>
		</Container>
	</div>
</div>
