import { Button, useMessage } from 'artific-ui'

export const Template = args => {
    const message = useMessage()
    message.config({
        dismissible: true,
    })
    return (
        <>
            <Button
                onClick={() => {
                    message.success('Hi ! This is a message').hold()
                }}
            >
                Success
            </Button>
            <Button
                onClick={() => {
                    message.info('Hi ! This is a message').hold()
                }}
            >
                Info
            </Button>
            <Button
                onClick={() => {
                    message.error('Hi ! This is a message').hold()
                }}
            >
                Danger
            </Button>
            <Button
                onClick={() => {
                    message.warning('Hi ! This is a message').hold()
                }}
            >
                Warning
            </Button>
        </>
    )
}

# Messages

The useMessage is like an message box that is only shown for a couple of seconds when something happens (i.e. when the user clicks on a button, submits a form, etc.).

<div>
    <div name="Default">{Template.bind({})}</div>
</div>

## API

```jsx
import { Message } from 'artific-ui'
```

<b>useMessage()</b> instance has following methods to use:

`config()` to set the config and supported props to the loader.

`(success|info|error|warning)()` method is used to show the message with their respective color schemes.

`alert()` method is used to the message with the predefine library themes. ie. (`primary`,`secondary`,`dark`,`light`)

`hold()` chain the hold method to your message to make message no disappear after the specific time.

export const SizeTemplate = args => {
    const sizes = ['xl', 'lg', 'md', 'sm', 'xs']
    const btns = sizes.map((v, i) => {
        const message = useMessage()
        message.config({
            size: v,
            dismissible: true,
        })
        return (
            <div key={i}>
                <Button
                    onClick={() => {
                        message.success('Hi ! This is a message').hold()
                    }}
                >
                    Message `{v}`
                </Button>
                <br />
            </div>
        )
    })
    return <>{btns}</>
}

# Message Sizes

you can use a `size` prop to define the size of your components

Set `size` prop in config with the value of (`xs`|`sm`|`md`|`lg`,`xl`). you can mix them with different supported `theme` or `layout`

Note: Mostly all the components have the same size values ie.(`xs`|`sm`|`md`|`lg`,`xl`). but some of the components might have additional sizes.
