# Dropdown

A dropdown menu is a toggleable menu that allows the user to choose one value from a predefined list

## Basic Dropdown

<div>
    <div name="Basic Dropdown">{Template.bind({})}</div>
</div>

## API

```jsx
import { Dropdown } from 'artific-ui/Dropdown'
```

## Dropdown Themes

If you want to use a different theme for `Dropdown` use the `theme` prop to set the theme
Note: All components have basic four themes ie. (`primary`|`secondary`|`dark`|`light`). you can use either of them.

`colorScheme` props to define your color scheme.

colorScheme is an object containing three properties: `background`, `backgroundColor`, `color`
